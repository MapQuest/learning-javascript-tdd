How to Test Your Javascript
(An Introduction to Jasmine)

So you're no longer just adding little bits of interactivity to your page, you're creating full blown applications in the browser.

You suddenly find:

  • You have several thousand lines of javascript.
  • You've developed reusable pieces of code.
  • You're still manually clicking through your web app to make sure everything works.

It's time to give your client side code the same love as your server code. You'll learn to use tools to test your javascript, automate the tests, and integrate them into your build system.

While this is not a comprehensive guide to Jasmine, it provides the basics for getting started, and should get you started on the path to having a comprehensive test suite for your client side code.
